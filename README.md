# rss parser
Parse rss data into json format using python

## Installation

Fork / clone the repo 

## Usage

- download rss from url add into forder to replace rss.xml 

- replace mytree in rss_element_tree.py with new rss filename

- click and run


## Dependancies
Rely on element_tree and json
